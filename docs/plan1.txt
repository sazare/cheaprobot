・体内の反応
　苦、快、

・体内の反応はセンサによる観測より高速。
　　苦・快・無
　　苦・快が動作を開始させるとすると、「無」はなくてもいい

・苦は消滅させる
・快はそこに向かう
・無は観測・学習を続ける

・つまり、学習が本業

活動
　苦の種類
　　・空腹(血中糖度の低下?)
　　・全身のどこかに対する痛(触か?)
　　
　　・尿意(なんらかの化学反応?)
　　・便意(なんらかの化学反応?)

　活動(Action Card)
　　・全身に対する緊張(close)
　　・全身に対する弛緩(open)

　　ここで「全身」しかないのは、幼児がそれ以上細分化できないから。
　　　(身体を細分化できれば、より細かくopen/closeができるようになる。)
　　知覚については、この段階では、聴覚のみ発達している。
　　　聴覚の学習を手本に他の知覚も発達するはず。

　　　皮膚感覚はあると思うが、細分化されていない。

　例 空腹
　　空腹 → 反応:苦 → 全身の緊張 = 泣く
　　外部からの乳房の出現
　　吸う(噛み付く、吸う)という行動はどこで覚えたのか??
　　{体:血中糖度の向上} → 反応:苦 の消滅 → 緊張の解除 → 泣き止む

　例 排便
　　便意 → 反応:苦 → 全身の緊張 → 泣く
　　　ここで、全身緊張から腹部緊張に偶然身体反応をつかまえると、排便がおこるだろうが、そこにいたるまでには時間がかかる。
　　　だから、排便は勝手にやる。
　　身体:尻の違和感 → 反応:苦 → 全身の緊張 → 泣く

　　　つまり、排便の結果、おしりのあたりの違和感がうまれ、これを苦と感じて泣くという仮説

　例 排尿
　　これは、あまり泣く理由がない。
　　そもそも胎内では、全身が液体の中にあったので、お尻が濡れることに違和感はないはず。
　　だから、それを苦と感じないので、泣く必要はない。

　　ということは、赤ん坊が泣くのは、排便したとき。
　　排便の前ではないはず。

★赤ん坊の活動は
　1) 空腹と授乳
　2) 排便(排尿)
